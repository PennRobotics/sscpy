# Stardew Multiplayer Seed Cracker

This is my own effort to simulate geode cracking in order to determine multiplayer ID, which is an internal variable not accessible to Switch 2 users. The multiplayer ID is also used for mystery boxes, night events, and Feast of the Winter Star.
